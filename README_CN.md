## 功能特点

- **支持 11 种语言** - 西班牙语、法语、德语、意大利语、葡萄牙语、日语、韩语、中文、阿拉伯语、俄语和印地语
- **简洁优雅的设计** - 奶油色背景搭配衬线字体，无干扰学习体验
- **离线可用** - 所有词汇本地存储，无需网络
- **智能分级系统** - 所有语言支持 CEFR 等级（A1-C2），日语额外支持 JLPT 等级（N5-N1）
